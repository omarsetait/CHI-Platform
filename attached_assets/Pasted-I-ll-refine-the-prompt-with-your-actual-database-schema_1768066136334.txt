## Feature Request: Dream Reports - View Related Claims Functionality

### Context
Enhance the Dream Reports > Generate Report page to add comprehensive claim viewing and export capabilities for findings across multiple workflows using the existing TachyHealth database schema.

---

### Database Schema Reference

**Core Tables:**
- `dream_reports` - Main reports with findings stored in JSONB
- `operational_findings_ledger` - Findings from Claims Management and Audit workflows
- `fwa_cases` - FWA case management
- `fwa_analysis_findings` - FWA findings with evidence
- `fwa_categories` - FWA categorization results
- `claims` - Legacy claims data
- `fwa_claim_services` - Service-level claim details
- `fwa_clinical_documentation` - Clinical documentation per claim

- Filter: `module_source = 'claims'` OR `finding_type IN ('claim_pattern', 'utilization_anomaly')`

- Primary source: `fwa_cases` joined with `fwa_analysis_findings`
- Alternative: `operational_findings_ledger` where `module_source = 'fwa'`
- Claim IDs: `fwa_cases.claim_id` or `fwa_analysis_findings` evidence

Each "View Related Claims" button must filter claims based on the finding's source and type.

async function generateAIReport(findingId) {

### Database Query Examples for Reference

**Get Dream Report Claims:**
```sql
SELECT c.*, fcs.service_code, fcs.service_description, fcs.total_price
FROM dream_reports dr,
     jsonb_array_elements(dr.findings) as finding,
     jsonb_array_elements_text(finding->'claim_ids') as claim_id
JOIN claims c ON c.id = claim_id
LEFT JOIN fwa_claim_services fcs ON fcs.claim_id = c.id
WHERE dr.id = $1 AND finding->>'id' = $2
```

**Get Operational Finding Claims:**
```sql
SELECT c.*, fcs.*
FROM operational_findings_ledger ofl,
     unnest(ofl.claim_ids) as claim_id
JOIN claims c ON c.id = claim_id
LEFT JOIN fwa_claim_services fcs ON fcs.claim_id = c.id
WHERE ofl.id = $1
```

**Get FWA Case Claims:**
```sql
SELECT 
  fc.claim_id,
  c.*,
  fcs.*,
  faf.finding_type,
  faf.severity,
  fcat.category_type
FROM fwa_cases fc
JOIN claims c ON c.id = fc.claim_id
LEFT JOIN fwa_claim_services fcs ON fcs.claim_id = fc.claim_id
LEFT JOIN fwa_analysis_findings faf ON faf.case_id = fc.id
LEFT JOIN fwa_categories fcat ON fcat.case_id = fc.id
WHERE fc.id = $1
```
